Cyber Security Risk Assessment System
 Project Description

This project is developed as part of the course “Auditing IT Infrastructure for IT Compliance”.

It is a web-based application designed to perform IT security audits, assess risks, and generate detailed reports to support compliance evaluation.

🎯 Objectives
To simulate real-world IT audit processes
To evaluate system security posture
To identify risks and provide recommendations
To visualize audit results effectively
⚙️ Features
🔐 Authentication using JWT
👥 Role-Based Access Control (Admin / Client)
📋 Multi-section Audit Form (11 sections, 40+ questions)
📊 Weighted Risk Scoring System (0–100)
🚦 Risk Levels (Low, Medium, High, Critical)
💡 Automated Recommendations
📈 Dashboard with Charts & Risk Meter
📄 PDF Report Generation
🕓 Audit History Tracking
🛠️ Admin Panel with Filtering
🧠 Concepts Implemented
IT Infrastructure Auditing
Risk Assessment & Analysis
Compliance Evaluation
Secure System Design
Data Visualization

User logs in
Performs audit by answering questions

Risk level is determined
